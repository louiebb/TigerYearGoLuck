## 其他

### currentColor

- 代表了当前元素被应用上的color颜色值

- 例子

```html
    <div style="color:blue; border: 1px dashed currentColor;">
        The color of this text is blue.
    <div style="background:currentColor; height:9px;"></div>
        This block is surrounded by a blue border.
    </div>
```
